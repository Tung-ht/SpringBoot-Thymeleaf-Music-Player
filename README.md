# Xpotify
Music player + Admin Dashboard

### Technologies Used
* Spring Boot
* Spring OAuth2 + Form login, Authentication and Authorization
* Spring Data JPA
* MySQL
* Thymeleaf
* Maven

### Prerequisites
Java 1.8 or greater, Spring boot 2.0 or greater, Maven, IntelliJ or Eclipse
